# DoctorMX - Technology Stack

## Frontend Framework
- **React 18.2.0** with TypeScript 5.2.2
- **Vite** for build tooling and development server
- **React Router Dom 6.16.0** for routing

## Styling & UI
- **Tailwind CSS 3.3.3** for utility-first styling
- **Framer Motion** for animations
- **Lucide React** for icons
- Custom design system with brand colors (jade, sky, sun)
- Responsive design with custom breakpoints

## State Management
- **XState 4.38.2** for complex state machines
- **React Context API** for application state
- **@tanstack/react-query 5.28.4** for server state management

## Backend & Database
- **Supabase** for PostgreSQL database, authentication, and storage
- **Netlify Functions** for serverless backend

## AI & ML Services
- **OpenAI API** for GPT models
- **Claude integration** for AI responses
- Medical image analysis capabilities

## Development Tools
- **TypeScript** with strict mode enabled
- **ESLint** with React hooks and refresh plugins
- **Playwright** for end-to-end testing
- **Autoprefixer** and **PostCSS** for CSS processing

## Deployment & PWA
- **Netlify** for hosting and serverless functions
- **Service Workers** for offline functionality
- **Manifest.json** for PWA features

## Internationalization
- **i18next** with browser language detection
- **react-i18next** for React integration
- Spanish (es) and English (en) support
