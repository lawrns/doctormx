# DoctorMX - Project Overview

## Purpose
DoctorMX is a comprehensive medical AI platform providing virtual consultations, symptom analysis, and healthcare services specifically designed for the Mexican market. The platform features:

- Advanced AI doctor capabilities using OpenAI GPT models and Claude integration
- Interactive chat interface for medical consultations
- Appointment scheduling and prescription management
- Cultural context awareness for Mexican healthcare
- Progressive Web App (PWA) with offline functionality
- Medical image analysis capabilities
- Herb recommendations and holistic health services

## Target Market
- Mexican healthcare market
- Cultural sensitivity in AI responses
- Compliance with Mexican healthcare regulations
- Spanish language support with i18n integration

## Key Features
- AI Doctor virtual consultations
- Symptom questionnaire and analysis
- Medical image capture and analysis
- Appointment booking system
- Doctor recruitment and registration
- Progress tracking and health monitoring
- Constitutional analysis and herb recommendations
- Subscription management system
