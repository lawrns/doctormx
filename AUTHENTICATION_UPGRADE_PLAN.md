# DoctorMX Authentication & User Tracking Upgrade Plan

## Executive Summary
Transform DoctorMX from a session-based anonymous service to a comprehensive authenticated health platform that tracks patient journeys, medical histories, and provides personalized care continuity.

## Current State Analysis
- **Session-based tracking**: Limited to browser session
- **No data persistence**: Conversations lost on refresh
- **No personalization**: Each visit starts fresh
- **Auth system exists but inactive**: Supabase infrastructure present but unused

#### 2.2 Unified Conversation Service
```typescript
// New service to manage conversations with proper context
export class UnifiedConversationService {
  private userId: string | null;
  private sessionId: string;
  private conversationState: ConversationState;
  
  async processMessage(userMessage: string): Promise<UnifiedResponse> {
    // 1. Update conversation state
    const context = await this.buildFullContext(userMessage);
    
    // 2. Generate AI response WITH contextual options
    const aiResponse = await this.generateUnifiedResponse(context);
    
    // 3. Store in database if authenticated
    if (this.userId) {
      await this.persistMessage(userMessage, aiResponse);
    }
    
    // 4. Return unified response
    return {
      text: aiResponse.text,
      answerOptions: aiResponse.contextualOptions, // Generated by same AI
      metadata: aiResponse.metadata
    };
  }
  
  private async buildFullContext(userMessage: string): Promise<EnhancedContext> {
    return {
      currentMessage: userMessage,
      conversationHistory: this.conversationState.messages,
      answeredQuestions: this.conversationState.answeredQuestions,
      userProfile: await this.loadUserProfile(),
      medicalHistory: await this.loadRelevantMedicalHistory(),
      previousConsultations: await this.loadSimilarConsultations()
    };
  }
}
```

#### 5.2 Data Migration
```typescript
export class SessionToUserMigration {
  async migrateAnonymousSession(sessionId: string, userId: string): Promise<void> {
    // Migrate current session
    const session = await this.loadSession(sessionId);
    await this.assignSessionToUser(session, userId);
    
    // Offer to import recent sessions
    const recentSessions = await this.findRecentSessions();
    if (recentSessions.length > 0) {
      const imported = await this.promptSessionImport(recentSessions);
      await this.mergeSessions(imported, userId);
    }
    
    // Generate health insights from imported data
    await this.generateInitialHealthProfile(userId);
  }
}
```
